# Dual-Dice

It is an engaging game where there are two virtual dice as dice 1 and dice 2 face off in a battle of luck! Using the Math.random() method, each die randomly generates a number, and the competition is on. If the number on Die 1 is higher, Player 1 wins else if Die 2 rolls higher, Player 2 is the winner.If the both the die rolls equal number then it's a tie.
